# Bootstrap Website

This project is a responsive website developed using **Bootstrap 5** as part of my internship task.

## Project Overview
The website is designed for a fictional restaurant and built using Bootstrap components to create a modern, responsive, and user-friendly interface.

## Features
- Responsive Navigation Bar
- Hero/Content Section
- Card Layout for Menu Items
- Responsive Design for Mobile & Desktop
- Footer Section
- Interactive Buttons

## Bootstrap Components Used
- Navbar
- Container
- Grid System (Row & Columns)
- Cards
- Buttons
- Responsive Layout

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Purpose
The purpose of this project is to learn and implement **Bootstrap 5 classes and components** to build responsive websites efficiently with minimal custom CSS.
